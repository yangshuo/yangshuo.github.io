简单的排序算法

> 冒泡排序

```java
    public void swap(int[] datas, int i, int j){
        int temp =  datas[i];
        datas[i] = datas[j];
        datas[j] = temp;

    }
    public void bubbleSort(int[] datas){
        for(int i = datas.length; i > 0; i--){
            for(int j = 1; j < i; j++){
                if(datas[i] < datas[i - 1]){
                    swap(datas, i, j);
                }
            }
        }
    }
```
